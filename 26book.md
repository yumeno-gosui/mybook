## INTRODUCTION

ボドゲバーで一緒に遊ぶ約束をした友人バンジャ君が、約束の時間になっても現れない。ひとまずお店に入ると、彼がいなくても食べ物を出して欲しい……と話が通っているらしい。

一体、何が起こっているのだろう？

あなたは、バンジャ君についてあれこれ考えてみることにしました。

----

本作は、友人抜きで予約された食べ物を食べることになった理由を解き明かそうとする、ソロミステリージャーナリングRPGです。ノート、オーディオレコーダー、ビデオ、ワープロ、コラージュなど...お好きな方法で調査を記録し、手がかりや出来事をつなげて物語を考え、ユニークでカスタマイズされたものにしましょう。物語は、あなたの想像力次第です。

### 感情的安全性

本作は、エンターテイメントと感情的な探求を提供することを目的としていますが、特定の人々にとっては嫌な記憶を引き出す引き金となる状況を扱うことがあります。もし不快に感じたり、ゲームを楽しめないと感じた場合は、自由にゲームから離れることができます。

再びゲームを遊びたくなったら、あなたがゲームから離れることになった要素を消したり、別人として物語を引き継ぐなどして、ゲームを再開してください。

プロンプトを変更したり、ランダムに頼らず選ぶなど、自分の好みに応じて感情の強さをいつでも調整することができます。創造的なプロセスとプロットへの没入を楽しみ、ミステリーを楽しんでください！

## あなたを作ろう

あなた（キャラクター）には2つの能力値があります：

- **【考察力】** 意図を読み取る、パターン認識、困難な状況の克服。作戦を練る
- **【感情度】** 共感的なつながり、気分を読む、感情的な対応、潜在的な危険を予測

あなたが一番得意だと思う能力値にD12、少し得意だと思うスキルにD10を割り振ります。

これらの能力値は、複雑な状況を打開する際の行動判定に利用します。【】の外にある項目は、能力値の使い方の一例です。実際には、能力値をもっと自由に解釈してみましょう。

### あなたは誰？

- あなたの名前は？
- バンジャ君とは、何時知り合いましたか？
- 一足先に食べ物を食べることになると知ったとき、どう感じましたか？
- この謎を解く原動力は何ですか？好奇心、愛情、それとも他の何か？
- 調査中、何を発見してしまうことを恐れていますか？

## ゲームの遊び方

### 行動判定

能力値を選択して行動判定する場合、その能力値に割り振られている現在のサイコロを2個振ります。
振った出目を確認し、大きい方の出目（ゾロ目なら好きな方）で下記の結果を確認します。

| 結果 | 判定結果 | 手がかり | 補足 |
| :---: | :---: | :---: | --- |
| 1〜2 | 判定失敗 | 手がかりを1つ減らす |判定に使用した能力値を減らす |
| 3〜4 | 悪ふざけ | 手がかりなし | 判定に使用した能力値を減らす |
| 5〜9 | 判定成功 | 手がかりを1つ得る |  |
| 10<br>以上 | 判定大成功 | 手がかりを2つ得る |  |

判定失敗：あなたの物語に何か悪いことが起こります。

悪ふざけ：何か予想外な騒動が起こって、手がかりかもしれないものを手に入れられません。

手がかりについては、《調査》の項目を確認してください。

----

行動判定によっては能力値を減らすように指示されたことがあります。その場合、以下のように小さい面のサイコロに取り替えることになります。

**[最大値] D12 → D10 → D8 → D6 → D4 → D2(コイン) [最小値]**


## セットアップ

本作は、3つのステージ《セットアップ》、《調査》、《推理》に分かれています。

《セットアップ》は、ゲームを遊ぶために必要なモノや材料を準備します。

### ゲームに必要なモノ

- 筆記用具
- ルール（本誌）
- ダイスセット（D4、D6、D8、D10、D12）を2つ
- D2用のコイン 2枚
- ジョーカーを抜いたトランプ1組：<br>スペードのスートだけをまとめた山札”スペードの山札”と、それ以外をまとめた山札”それ以外の山札”を作ってシャッフルしておく。

## 調査

本作は、3つのステージ《セットアップ》、《調査》、《推理》に分かれています。

《調査》のステージでは、各ロケーションを一度だけ調査できます。以下の質問事項を忘れないでください。

- バンジャはどこにいて、何故この場所に来ないのか？
- 出された食べ物に関して、複雑な思い出はあるのか？

すべてのロケーション調査するか、集めた手がかりに満足するまでラウンドを繰り返します。

### ラウンドのプレイ手順

各ラウンドは、以下のアクションを順番に行います：

1. 調査するロケーションを選ぶ。
2. D66を振ってコンプリケーション表を参照する。
3. コンプリケーションの内容に対して、能力値を1つ選んで行動判定を行う。
4. 行動判定の成否を元に、ロケーションや手がかりについて日記やメモを書く。
5. 次のロケーションを選ぶか、《推理》のステージを開始する。

最初のラウンドは、必ず[ボドゲバー]のロケーションからスタートします。コンプリケーション表を参照し、何が起こるか、最初の手がかりが得られる可能性があるかを確認しましょう。

### 手がかりについて

行動判定の結果が「判定失敗」「悪ふざけ」の際は、”それ以外の山札”の上からカードを1枚引いて裏向きで自分の前に置きます。
「判定成功」「判定大成功」の場合は、”それ以外の山札”から指定された枚数のカードをひいて表向きで自分の前に置き、後述する《手がかり表》を参照してください。

### 失敗からの学びについて
各ロケーションを調査するとき、行動判定中に1回だけ以下の特典を得られます。

**裏向きのカード1枚を捨札にすることで、行動判定で振ったサイコロ1つを振り直せる。**

捨札にしたカードは、山札に混ぜないようにしましょう。

### 休息

[ボドゲバー]のロケーションに戻って捜査の疲れを癒すことで、減少した能力値を1つだけ最大値にリセットすることができます。休息の様子は、日記に書いておきましょう。

《休息》を行うと、まだ行っていないロケーション1ヶ所を調査できなくなります。

### 転換

《転換》は、あなたが考える物語に刺激を与える要素となります。

以下のスケジュールに従って、転換を行います。

 1. ゲームスタート / [ボドゲバー]とロケーションを1つ調査する
 2. 転換1 / 食べ物を食べるか、スペードの山札から1枚引く
 3. ロケーションを2つ調査する。メモに記録するだけでも、日記を書いても良い
 4. 転換2 / 食べ物を食べるか、スペードの山札から1枚引く
 5. ロケーションを2つ調査する。メモに記録するだけでも、日記を書いても良い
 6. 仮説を立てる
 7. 転換3 / 食べ物を食べるか、スペードの山札から1枚引く
 8. その後について日記を書く。手紙でも良い

《転換》は食べ物を利用するか、スペードのオラクルを利用することが可能です。
どちらを利用するにしても、最初に”スペードの山札”から1枚カードを引きます。

#### 味のオラクル
転換のときに口にする食べ物を、

**本来、バンジャ君と食べるはずだった料理**

と見立て（ジュースをスープに、お菓子を料理として食べる）ることで、あなたの考えている物語にのめり込みやすくなります。味のオラクルは、口にした料理であなたが一番強く感じた味を採用します。味のオラクルは以下の通りです。

- **【甘味】-** バンジャ君との友情について
- **【塩味】-** バンジャ君とのポジティブな思い出
- **【酸味】-** 今回の状況についての悲しみ
- **【苦味】-** あなたが行動したことで感じた後悔
- **【うま味】-** もし、あなたが約束の場所に現れない側だったら？

味のオラクルを物語に取り入れる場合、スペードの山札から引いたカードは“表向き”で自分の前に置きます。
味のオラクルを物語に取り入れない場合、"スペードの山札"から引いたカードを捨札にします。捨札は、山札に混ぜないようにしましょう。

#### スペードのオラクル
スペード各オラクルは、以下の通りです。

| ランク | オラクル |
| :---: | --- |
| A | 考えている物語に新しい展開を加えること |
| 2 | 次に選ぶロケーションは、ゲームに設定されていないものにする |
| 3 | 考えている物語に転換点を加えること |
| 4 | 美しさを加えるアイテムやキャラクターを導入すること |
| 5 | 考えている物語に予期せぬ驚きや発見を導入すること |
| 6 | 物語に何かしら刺激を加える要素を導入すること |
| 7 | 物語に感情を揺さぶる出来事があったことを追加すること |
| 8 | 音楽に関連づけられた情報や出来事を追加すること |
| 9 | 物語に暗い影が忍び寄る表現を追加すること |
| 10 | 物語に落ち着かせる声の持ち主が何者だったのかを加えること |
| J | 物語に視覚的な魅力や意味を読み取る絵を見かけたことを加えること |
| Q | 物語に解決を持ちこ越した秘密や謎を付け加えること |
| K | 考えている物語に必ず本を登場させること |

オラクルの内容を物語に取り入れる場合、"スペードの山札"から引いたカードを“表向き”にして自分の前に置きます。
オラクルの内容を物語に取り入れない場合、"スペードの山札"から引いたカードを捨札にします。捨札は、山札に混ぜないようにしましょう。

オラクルを取り入れることによって、《推理》のステージが有利になります。


## 推理
本作は、3つのステージ《セットアップ》、《調査》、《推理》に分かれています。

《推理》のステージは、1回のゲームにつき1度だけプレイできます。一度でもこのステージを開始すると《調査》ステージへは戻れません。

《調査》ステージで見つけた手がかりをできるだけ多く使って、仮説を書き上げましょう。どの手がかりを使ったか、必ずわかるようにマーキングしてください。それらが終わったら、推理ロールを行います。

### 以下の質問の答えを盛り込もう

- バンジャはどこにいて、何故この場所に来ないのか？
- 出された食べ物に関して、複雑な思い出はあるのか？
- あなたの仮説が本当だったら、どうしますか？

### 推理ロール

使用するサイコロはD0(なし)からスタートします。仮説を作る際に使用した手がかりや転換の数だけ、サイコロを1段階ずつ大きくします。

**D0 → D2 → D4 → D6 → D8 → D10 → D12**

#### 逆転の奇策について
推理ロールの際に、どちらか1つの効果を1度だけ利用することができます。

****裏向きのカード1枚を捨札にすることで、行動判定で振ったサイコロ1つを振り直せる。****

または

**表向きのスペードの枚数分、推理ロールの結果に加算できる。**

推理ロールの結果が確定したら、以下の内容に当てはめてその後に何が起こったかを書き残します。

##### ⚫︎ 結果が3以下 - 失敗エンド

あなたの仮説は、完全に間違っている。我に返ると、店のマスターから話が通っていると再び聞かされる。出された料理を全て食べたはずなのに、目の前には何も置かれていなかった。しかも空腹を感じている。名前を呼ばれたので振り返ると、そこにはあなたが立っていた。

##### ⚫︎ 結果が4〜7 - 悪ふざけエンド

あなたの仮説は何処かが間違っており、その部分が悪ふざけや騒動が起こるキッカケとなる。

##### ⚫︎ 結果が8以上 - 成功エンド

あなたの仮説は、すべて正しかった。あなたはバンジャ君と合流でき、最後の食べ物に関する感想を言い合いながらボードゲームを楽しむことになる

## 手がかり一覧

トランプデッキから引いたカード1枚を表向きにした場合、スートを参照を参照して以下のキーワード群から1つ選びます。一度選んだキーワードは、再度選ぶことはできません。使用したキーワードを線で消すなどして区別を付けてください

自販機の珍しい商品、グルメリポーター、地元の話、奇妙な夢、夜間活動、失踪の噂、スナック菓子の誘惑、珍しい伝統、密会の噂、不快な表現、目撃談、共通の知り合い

#### クラブ

謎の足音、花の匂い、乾いたペンキ、証言、チョークの跡、血痕、設計図、散らばった小銭、壊れたアクセサリー、使用済みのチケット、落ちたボタン、残された飲み物のカップ
