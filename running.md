# ミッドナイト・スライド 〜夜を走る、言葉を紡ぐ〜


## イントロダクション
あなたは“何か事情を抱えて夜の街を走るランナー”であり、近場のミッドナイト専門のランニングコミュニティに所属しています。

このコミュニティに参加する絶対条件として、

**スポーツマスクの着用必須**

とあります。走りながら会話をしても、苦にはならないスピードでのランニング。でも、走りながらの雑談はほとんどありません。リズミカルに地面を蹴る音と、くぐもった呼吸だけが夜の空気に混ざり続けていました。
主催者以外どんな人達が参加しているかは謎なのに、今のあなたにとって居心地の良いコミュニティです。

ある日のこと。ランニングステーションで帰り支度を整え終えると、あなたの靴上に一通の手紙が置かれていることに気づきました。

**少しの間、文通しませんか？**

何故か見覚えがあるような気がする筆跡で、手紙には書かれています。
思うところがあるのか、それとも気まぐれか……。
手紙の返信を書こうと思い至って、この物語は動き始めました。


## はじめに
これは、身体を動かしながら4回の対話を試みる日常フィクションのランニング×半書簡形式ジャーナリングRPGです。プレイヤーが実際に走り（雑談ができるゆったりとしたスピードで、プレイリストを聴きながら）、実際に五感で取り込んだ情報を置き換え、あなたとして手紙（文章）を書くことになります。

本作は、「ホラー」「ミステリー」「恋愛」ジャンルではありません。世界の運命を揺るがすものではなく、あなたの周囲のみで起こる

**誰かと少しだけ生きる時間が重なる話**

ささやかな体験となります。


### ジャーナリングRPGについて
プレイヤーはあなた（キャラクター）の行動、性格、行動をコントロールします。提示されたプロンプトに答え、それを好きなように解釈し、自分の考えを記録していきます。自分の体験を記録するために、物理的な日記やスマートフォンなどを活用したデータ形式として利用することも可能です。人物や場所、興味をそそる状況を創作し、過去の出来事と現在のプロンプトの間につながりを形成して、好きなだけ書いみましょう。

#### 半書簡形式について
あなたはこれから4通の手紙を読み、ゲームとして返事を書くことになります。ですが、あなたがどのような内容を手紙に込めても、次に読むであろう手紙はあらかじめ内容が決まっています。
あなたの感情、もしくは文脈から感じた解釈やギャップによって自由に物語を受け取ってください。

#### 遊ぶのに必要なもの
このゲームを遊ぶには、以下の物が必要です。
* 物語を考えたいプレイヤー1人
* 今日は走るぞーという気持ち
* 日記を記録するノート（またはブログやSNSサービス）
* 秒まで表示される、もしくは秒針まで備えている時計やスマートウォッチ

### テーブルでの安全
どのようなロールプレイングゲームをプレイする場合でも、あなたの安全と精神的健康が優先されるべきです。不快に感じたら、ゲームを中断してください。

新しいゲームを始める前に、あなたが遭遇したくない話題について、自分自身と確認したりする時間を取りましょう。定期的に自分自身をチェックし、自分の書いたものが自分にどのような影響を与えるかを評価しましょう。ランニングを継続させようとする魅力的なツールとなりますが、決して自分自身を罰するために使ってはいけません。

#### ライン&ベール
自分の欲求を伝える一つの方法です。ラインは、どんなことがあってもゲーム内で出てきてはいけないテーマです。ベールは、"カメラの外"でのみ起こるべきテーマです。もしあなたの手紙に、あなたが話し合っていないデリケートな話題が含まれているかもしれないと思ったら、それを約束する前に仲間に確認するのが常にベターです。TTRPG安全ツールキットには、このようなツールが含まれています。

#### コンテンツに関する警告
このゲームの世界構築では、感情が大きく揺さぶられることはありません。ですが、何かのはずみにネガティブな話題に触れる可能性はあります。これらのトピックに対して、常にライン＆ベールを念頭に置き、必要に応じて調整しながらアプローチを試みてください。このゲームでは、仲間に対する偏見、排他、人種差別、トランスフォビア、性差別、その他いかなる種類の差別を表現したいプレイヤーや、いかなる形の憎悪を広めたいプレイヤーのためのテキストではありません。

#### 創作形式の自由の保証
ジャーナリングRPGは、物語を記録する際に必ずアナログである必要はありません。
ブログやSNSサービスを利用しても良し、オンラインセッションツールを利用しても良し、音声ログでも良し、Vlogでも良し、文字を書かずに写真画像だけでも良し、誰かと雑談しながらでも……要は「遊んだ！」と満足できれば何だって良いのです。

*プレイヤー自身が、安心して楽しく遊べる方法を採用してください*

必要がありそうなら、TRPG安全ツールキットをご参照ください。
https://ttrpgsafetytoolkit.com/


## あなたについて
あなたは、夜にランニングする習慣にすっかり染まってしまいました。
よほどの悪天候出ない限り、週2〜3回は走っています。
ランニングをすることに、何か理由があるようです。

以下の内容から【背景】を1つ選択します。

* **【選択肢A：帰れない人】** 家に帰りたくなくて、あえて遠回りして走っている
* **【選択肢B：探している人】** 逃げた猫、または失くした何かを探す口実で走っている
* **【選択肢C：リセットしたい人】** 今日あった出来事を、汗と一緒にすべて流したくて走っている

### あなたに成るプレイヤーについて
現実でランニングをすることになるプレイヤーは、走りながら雑談しても苦にならないペース（130BPM前後）で走ったデータをスマートウォッチで記録することになります。
プレイリストなども利用して、ペースやBPMを管理することも大歓迎です。

#### マスクについて
本作における“スポーツマスク”は、プレイヤーからあなたに成りきるためのスイッチ的な役割となります。
ですが、通販以外での入手が難しくなってきました。
そのため、ランニング時に必ず着用する小物（お気に入りのキャップ、スマートウォッチのバンド、特定のイヤホンなど）を、“スポーツマスク”の替わりにスイッチ的な役割として利用してみてください。


## ゲームの流れ
### Phase 1：リーデイング（準備）
あなたへ届いた手紙を読み、あなたとしてランニングの準備をする。

### Phase 2：ランニング＆オブザベーション（観察）
夜の街へ走り出します。あなたの五感で夜の街を感じ、「夜の街のディテール」へと目を向けます。

### Phase 3：ランニング＆メモリー（記憶）
「ここからペースを上げていい」と感じた瞬間に、スマートウォッチを確認して“分”と“秒”を一度だけ確認して暗記します。

### Phase 4：ライティング（独白）
帰宅後にクールダウンを済ませてから、感じた&暗記した内容を手がかりに手紙の返信を書きます。


そうして、次の手紙があなたに届きます。

### Ending
4通目の手紙の返信を書き終えたら、ゲームは終了となります。

封筒に入っていたり、丁寧に折られていたり……見た目は様々でした。

ささやかな好奇心に負けて返事を書くことにしたあなたは、手紙の筆跡に見覚えがあることを不思議に思いながら、読み返します。

### Phase 2：ランニング＆オブザベーション（観察）
あなたは手紙の内容を頭の中で反芻しながら、スマートウォッチの時間を確認します。
「分」の値を利用して、ランニング中に何を意識して走れば良いのかを決める「チェックポイント表」を参照します。
参照した候補キーワードから、【チェックポイント】を1つを選択します。

#### チェックポイント表
| 分の値 | カテゴリー | 候補キーワード |
| :----: | :---- | :---- |
| 00〜14 | 光 | 一番明るかったもの、一番暗かった場所、揺れていた光、消えそうだった光、足元を照らしていた光、一度だけ見えた光 |
| 15〜29 | 音 | 一番近かった音、一番遠かった音、自分しか聞いていない気がした音、心地よかった音、少し気になった音、夜らしいと思った音 |
| 30〜44 | 匂い | 季節を感じた匂い、足を止めたくなった匂い、懐かしい匂い、今日だけの匂い、どこから来たかわからない匂い、帰り道を思い出した匂い |
| 45〜59 | 人 | 一番印象に残った人、目が合った人、声だけ聞こえた人、誰かを待っていた人、急いでいた人、誰もいなかった場所 |

### Phase 3：ランニング＆メモリー（記憶）
いよいよ、夜の街をゆっくりと走り始めます。
信号で立ち止まったり、プレイリストの曲の変わり目が訪れたり、【チェックポイント】に遭遇するたびに、手紙の内容に盛り込めそうな「夜の街のディテール」を感じようと試みます。

#### 夜の街のディティールとは
それは、走っている風景を【チェックポイント】に注意しながら観察することから始まります。
* 観察によって得た印象を視覚・嗅覚・聴覚などを織り交ぜて表現します。
* **「孤独だけど心地いい」** 感情を入れる（ただ綺麗じゃなく、心理的な魅力も伝える）
* **「走るからこそ」** を強調（散歩や車とは違う理由を伝える）
* 具体的な小ネタを1個入れる（水溜まりの反射、足音の反響など）
* あまり、不思議な体験にしないこと。

ランニング中、ここから少しペースを上げていいと感じた瞬間(BPMが140〜のプレイリストに切り替えたいなど)にスマートウォッチを確認して“分”と“秒”を一度だけ確認して暗記します。
この情報に関して、紙やデバイスに記録せずに暗記する様に心がけましょう。

### Phase 4：ライティング（独白）
帰宅してクールダウンした後に、日記を書く準備をしてスマートウォッチのデータの同期が完了したら、このフェースが始まります。

#### 4-1.記憶した内容をメモする
ランニング中に暗記した“分”と“秒”を、このタイミングでメモします。
この2つの値は、手紙を書くための【プロンプト】となります。

#### 4-2.手紙の内容の素材を膨らませる
あなたがランニング中で観察した「夜の街のディテール」を、【プロンプト】を使って内容を膨らませていきます。

##### プロンプト表（イベント）
| 分の値 | 意味・印象 | 具体的な例 |
| :----: | :---- | :---- |
| 00〜14 | 見つける | 今夜見つけたものを一つ書く。それは物でも、景色でも、感情でもよい |
| 15〜29 | 待つ | 誰か、あるいは何かを待っていた記憶を書く。今も待っていることでもよい |
| 30〜44 | 忘れる | 忘れようとしていること、忘れてしまったこと、思い出せないことを書く |
| 45〜59 | 渡す | 誰かに渡したかった言葉、渡せなかったもの、あるいは受け取ったものを書く |

##### プロンプト表（感情）
| 秒の値or分の値 | 感情の種類 | 具体的な例 |
| :----: | :---- | :---- |
| 00〜14 | 安心 | 少し肩の力が抜けた調子で書く。相手を責めず、肯定するように |
| 15〜29 | ためらい | 書こうか迷っていることを少しだけ書く |
| 30〜44 | 後悔 | 「もしあのとき」を一度だけ考えるように |
| 45〜59 | 期待 | 明日のこと、次に走る夜のこと、まだ起きていないことを少しだけ書く |
※ 4-2では“秒”の値、4-4では“分”の値で参照します。

##### もし、暗記した内容を忘れてしまった場合
ランニング中に記憶したはずの“分”と“秒”を思い出せない場合、次の内容から任意に1つ選びます。
その内容に沿って「夜の街のディテール」の内容を膨らませていきましょう。

* 曖昧（見た気がするが思い出せない）：手紙に「何か書きたいことがあったはずなのに、思い出せません。夜の光のせいにさせてください」と書き添える。
* 不在（その時間だけ記憶が抜けている）：手紙の一部を、あえてボールペンで黒く塗り潰すか、空白のままにする。
* 重複（同じルートを2回走った気がする）：同じ単語や同じフレーズを、手紙の中で何度も繰り返して書いてしまう。

#### 4-3.手紙を書く
「夜の街のディテール」の内容を膨らませられたら、「Phase 1」で読んだ内容を今一度読み返します。
その後で、あなたなりの手紙の返信を書きます。

#### 4-4.手紙を出す
書いた手紙は、プレイヤー宛に届くようにポストへ投函します。または、プレイヤーが必ず中を確認する郵便受け（もしくは郵便受けに見立てた物）に入れておきます。
実際に自分宛に届くか、1〜2日時間を置いて自分で入れた手紙を取り出して、「Phase 1」の次の手紙へ進みます。

その際に、ランニング中に暗記した“秒”ではなく“分”の値を使って「プロンプト表（感情）」を参照します。
その内容が次の手紙に込められている“相手があなたに抱いているかもしれない感情”となります。
この感情を意識しながら、次の手紙を読んでみましょう。

### エンディング
手紙は書き終えましたが、あなたには考えていることが残っています。

**手紙の筆跡に見覚えがあること**

この筆跡を誰のものだと思っていましたか。
思い当たる人がいるなら、その人を一人だけ思い浮かべてください。
ですが、

**「……いや、やっぱり違うか」** とか　**「まぁ、いいんだけど」**

と思ったままで、この物語を閉じてください。



## アフターケア

何か飲み物だったりを用意して、「Phase 1」の4通と合わせて自分で書いた手紙を読み直します。

* 時々思い返してしまいそうな、フレーズ
* この物語を書きながら、現実のプレイヤーに当てはまりそうな気づき
* プレイヤーの目線で、あなたに伝えたいこと

これらを探してみてください。共有する必要はありません。
是非、内緒にしておきましょう。
