# Neon Crunch Foods

** 「今日の仕事のあと、何を食った？　その味はどんな気分を表してる？」**

>> ジャンル: サイバーパンク・ストリート・フードライフ・ジャーナリングRPG

あなたは、崩壊した日本のネオン廃墟区にある「Vat-Grown@maki」の屋台主としてストリートフードを売りながら、コーポのドローンが飛ぶ空の下で客を生かす日々を生き繋いでる。
定番は培養アジのオシニギリだが、オーガニックできっと本物の食材はいつでも求めていた。食べれば、きっと誰もが祖国の味を思い出す……はず。

## WHAT YOU NEED 
空腹を凌駕する想像力、ポーカーダイス（または6面体サイコロ）1つ、ペンと紙（もしくはそれに替わる記憶媒体）、トランプデッキ1組（ジョーカー含む）、ジョーカーの効果発現を管理するマーカー2色

## SETTING UP SHOP
屋台の場所をダイスで決めます（または自由に選択します）：
9(1)：廃墟区と新都市区の境界
10(2)：ストリートの暮らしを眺められる廃墟の2階
J(3)：暗渠の中へ降りられる工業用エレベーター
Q(4)：人工島の路上
K(5)：緑が暴走した植物園の前
A(6)：閉鎖された複合施設の最上階

## SETTING UP SHOP
日記の書き手であり、「Vat-Grown@maki」の屋台主であるあなたの設定を決めます。
設定の他に、あなたは能力値して【幸運値】が設定されています。初期値は「0」です。
設定を含めて決め終わったら、トランプデッキを良くシャッフルして上から5枚引いてひとまとめにします。
この5枚が「あなたの山札」です。それ以外は、以降「プロンプトの山札」と呼びます。

## PLAY OVERVIEW
1. 仕込みフェイズ [プロンプトA]
1d6を振り、出目と同じ枚数のカードを「プロンプトの山札」から引きます。
 + 内容：カードの数字やスートでプロンプト表を照らし合わせ、仕込み中に発生したイベントを1枚処理します。
 + 記録：処理したイベントの中で気になるキーワードをメモします。この段階で日記本文を書く必要はありません。あくまで、「キーワード」をメモします。
 + 判定：処理したイベントの結果がどうなるか、この後の接客フェイズ [プロンプトB]で客が食べるかもしれない「本日のオススメ」の出来栄えはどうかを行動判定に委ねます。
   + ポーカーダイス1個を振った結果と、「あなたの山札」の上から1枚引いたカードのランクを足します。最初の合計値が15未満の場合、「あなたの山札」から追加でカードを引いて、カードのランクを足せる（引いた場合は、必ず足すこと）。値が決定した後、幸運値を-1して成功を購入することができる。
   + 15（大成功）：最高の結果になる。幸運値+1。
   + 16〜21（成功）：十分な成果を得られる。
   + それ以外（不穏な成功）：成果はあるが、何かしら問題がつきまとう。
   + 22以上（致命的な失敗）：幸運値-1。
[プロンプトA]で使用したすべてのカードは「あなたの山札」に加えて、かならずシャッフルする。
2. 接客フェイズ [プロンプトB]
 + 訪問客:[プロンプトA]で振った1D6の出目を参照して、誰が屋台に訪れたのかを決定する。2回目以降は、この値を加算し続けます。
   - †：次の「日数」ロールの結果に2を加算します。
   - h：食べなかった方の料理を持ち帰ります。
   - 4：次の[プロンプトA]で振る1D6を2D6にして高い出目を採用する
 + 訪問客が店に来る前に体験した出来事：「プロンプトの山札」の上から1枚引きます。
 + 内容：カードの数字やスートでプロンプト表を照らし合わせ、訪問客が店に来る前に体験した出来事を処理します。
 + 記録：処理したイベントの中で気になるキーワードをメモします。日記は書かず、あくまで「キーワード」をメモします。
 + 判定：処理したイベントの結果がどうなるか、決められない場合は行動判定を行います。
 + 後処理：[プロンプトB]で使用したすべてのカードも「あなたの山札」に加えます。「あなたの山札」にある数字カード5枚を「プロンプトの山札」へ戻して、絵札1枚を加えることできます。どちらの山札も忘れずにシャッフルしてください。
 + オラクル：訪問客が「定番」メニューを食べるのか、「本日のオススメ」メニューを食べるのかをオラクルによって決めます。
   1. 「あなたの山札」の上から「3 + 幸運値」枚分を公開する。
     + 絵札が1枚以上：オラクルの結果は、「はい」。訪問客は「本日のおすすめ」を食べます。絵札の枚数が多いほど、客の満足度が高くなる。
     + 絵札なし：オラクルの結果は、「いいえ」。訪問客は定番メニューである培養マグロのヅケニギリを食べます。
     + 7のカード：7のカードを「プロンプトの山札」へ戻し、コラージュした自作絵札を1枚作成して「あなたの山札」に加えて山札をシャッフルします。
      - 幸運値が「+」の場合：日記からポジティブな3語を選び、コラージュしたカードの解釈とする。次以降引き当てたら、日記の内容にポジティブさを増やせる。
      - 幸運値が「0」か「-」の場合：日記からネガティブな3語を選び、コラージュしたカードの解釈とする。次以降引き当てたら、日記の内容にポジティブさを増やせる。
3. 閉店・日記フェイズ
一日の終わりに、客との会話や食べた反応、街の様子、扱った食材やニギリの感覚を日記に記します。自作絵札の説明もここで書き留めます。

## END GAME
「24. ？？？」を接客し終えるか、自作絵札を7枚作った時点であなたは選択を迫られます。ここで営業をつづけるのか？それとも新天地で新たな一歩を踏み出すのか。最後の日記を書いた際の幸運値によって今後の方向性が変わることになります。
 * 幸運値が+：屋台は繁盛し、店舗を構える話が舞い込んでくるかもしれません。
 * 幸運値が0：現状の日々が繰り返します。でも、もしかしたらどこかが違うかもしれません。
 * 幸運値が-：屋台が壊されるか、存続するために危ない仕事も引き受けることになりそうです。

これまでに作った自作絵札のキーワードを絡めながら、日記の最後の一行にどうなるか結末を書き加えます。

### ジョーカー
本作でのジョーカーは、物語にかなりの混乱を与えることになります。
* ジョーカーを引いた場合、その日のイベントをすべて破棄してジョーカーのプロンプトを優先させます。
* 「幸運値」の±が反転します。0なら何も起こりません。
* ジョーカーも、「あなたの山札」に加わります。ジョーカーの効果をリセットするには、もう一度ジョーカーを引き当てるか、自作絵札1枚の内容を反転させる(ポジティブ⇔ネガティブ)必要があります。

| ジョーカーの色 | 効果 | 内容 |
|:-----------|:------------:|:------------|
| 赤 | ノイズ | 街にノイズが走り、ドローンが謎の統制を見せ、ネオンが瞬断し、停電などが起こる。復旧しても街のルールが上書きされる。1d6を振った際、「6-出目」の値を利用する |
| 黒 | リセット | あなたの記憶が消えたり、噂やニュースが一気に広まり、見覚えのある客が“別人”として現れたりする。ゲーム中、「あなたの山札」の下からカードを引くことになる |
